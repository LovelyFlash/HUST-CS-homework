任务描述
本关任务：编写程序实现双向栈。
假定以顺序存储结构实现一个双向栈，即在一个一维数组的存储空间中存在着两个栈，它们的栈底分别设在数组的两个端点。试编写实现着个双向栈的3个函数：
初始化栈inistack、入栈push和出栈pop
