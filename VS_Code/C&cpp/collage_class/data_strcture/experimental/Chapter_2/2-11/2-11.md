任务描述
本关任务：编写一个函数实现线性表的删除操作。
函数原型：status ListDelete(LinkList &L,int i,ElemType &e)
功能说明：若线性表L不存在，返回INFEASIBLE；否则删除线性表L的第i个元素，若删除成功则将删除的值赋给e并返回OK，若删除失败则返回ERROR。
